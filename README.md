# Healthcare Insurance Claim Approval Agent 
## A Healthcare Insurance Claim Approval Agent that automates claim adjudication by using LLM reasoning and data validation.
### What was created?
A React-style agent that:
- Summarized patient records
- Summarized policies
- Evaluated arguments against the policies
- Made clear, explainable decisions for approvals or routes for review
### Highlights
- Used LangGraph to manage the workflow
- Created tools for the agent to use when generating responses
- Achieved over 80% precision and recall
Unfortunately, this project was created in a secure Databricks environment, which means it can no longer run.
